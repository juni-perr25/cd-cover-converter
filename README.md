## Project Description
This project allows you to upload images to composite them with a background using the darken blend mode. The application provides a batch image processing feature that optimizes and enhances your images.

## Setup Instructions
To get started, clone the repository and install the dependencies:

```bash
npm install
```

After installation, run the server:

```bash
npm start
```
